# 📋 Планировщик задач

Персональный планировщик задач с Telegram-ботом, AI-категоризацией через Qwen и веб-интерфейсом.

## Возможности

- ✅ Создание задач через Telegram или веб-интерфейс
- ✅ AI-категоризация задач (Qwen через HuggingFace)
- ✅ Распознавание скриншотов Яндекс.Календаря
- ✅ Управление подзадачами
- ✅ Периодические задачи с расписанием
- ✅ AI-анализ нагрузки и рекомендации
- ✅ Архив задач
- ✅ Просмотр задач на любую дату
- ✅ Синхронизация пропущенных из Telegram при включении
- ✅ Перенос невыполненных задач + анализ хронических

## Быстрый старт

### 1. Установка зависимостей

```bash

./setup.sh
```


### 3. Запуск

```bash
./run.sh
```

### 4. Открыть

- **Веб-интерфейс:** http://localhost:8000
- **API документация:** http://localhost:8000/docs
- **Telegram бот:** найдите вашего бота в Telegram

## Структура проекта

```
plan/
├── app/
│   ├── api/          # API эндпоинты (tasks, categories, ai, screenshot)
│   ├── bot/          # Telegram бот (aiogram)
│   ├── db/           # База данных (database, seed)
│   ├── models/       # SQLAlchemy модели
│   ├── services/     # Бизнес-логика (ai, ocr, rollover, recurring)
│   ├── web/          # Веб-интерфейс (Jinja2 + HTMX + Tailwind)
│   ├── config.py     # Настройки из .env
│   └── main.py       # FastAPI приложение
├── config/           # Контекст AI + feedback лог
├── docs/             # Спецификация и план
├── tests/            # Тесты
├── uploads/          # Загруженные скриншоты
├── .env              # Токены (не коммитится)
├── setup.sh          # Скрипт установки
└── run.sh            # Скрипт запуска
```

## Категории

Категории настраиваются в `config/categories_context.md`:

```

├── Финансы, Документы, Проекты, Инфраструктура, Контент

🏠 Личное
├── Семья, Здоровье, Финансы, Цифровая гигиена, Покупки, Социум, Пет-проекты, Свои сайты

├── Курсы, Инструменты, Контент
```
